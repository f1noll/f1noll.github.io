# LR-2 — Основы NumPy

## Описание задачи
В лабораторной работе необходимо было реализовать функции для работы с массивами и данными с использованием библиотеки NumPy.  
Функции должны соответствовать стандарту PEP-8.

## Решение
В работе были реализованы функции для:

- создания и преобразования массивов NumPy;
- выполнения векторных операций;
- выполнения матричных операций;
- вычисления статистических показателей;
- построения графиков.

## Нюансы
Большинство операций уже реализованы в библиотеках, поэтому важно было корректно применять соответствующие функции.
Также необходимо было соблюдать требования к оформлению кода: добавлять аннотации типов, писать docstring для функций и придерживаться стиля кода согласно стандарту PEP-8.

## Результат
Все тесты успешно проходят.  
Код содержит аннотации типов и документацию.
